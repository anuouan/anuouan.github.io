## 2. MkDocs介绍

根据[MkDocs](https://www.mkdocs.org/)官网介绍：

MkDocs是一个**快速**、**简单**、**华丽**的静态站点生成器，适用于构建项目文档。文档源文件使用Markdown编写，并使用单个YAML配置文件进行配置。

MkDocs的特点有：

- 多个漂亮主题
- 易于个性化
- 实时预览
- 随处托管
